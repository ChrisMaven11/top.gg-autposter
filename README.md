# Installation

`npm install @top-gg/sdk` or `yarn add @top-gg/sdk`

# Posting Bot Stats

`npm i topgg-autoposter`

Include it in your bot's file, and type `node index.js` 

# Refresh Data
Click on the edit button **--->** Save Changes **--->** Refresh data
